# pbo-booking-system
Final Project PBO - Booking System using Python OOP
## Deskripsi Proyek
Proyek ini merupakan Sistem Reservasi (Booking System) yang dibuat menggunakan bahasa Python
dengan konsep Pemrograman Berorientasi Objek (PBO).
Sistem ini memungkinkan pengguna untuk melakukan pemesanan dan mengelola data reservasi.
## Tujuan Proyek
- Menerapkan konsep Pemrograman Berorientasi Objek (PBO)
- Membuat sistem reservasi sederhana
- Mengelola data pemesanan secara terstruktur
## Konsep PBO yang Digunakan
- Class dan Object
- Encapsulation
- Inheritance (jika ada)
- Method dan Attribute
## Cara Menjalankan Program
1. Pastikan Python sudah terinstall
2. Buka folder project
3. Jalankan file utama:
```bash
python main.py
